Prescription prepayment certificates

Buy a prescription prepayment certificate

Buying a prepayment certificates can help you save money if you need to get repeat prescriptions.

A one off prescription costs £9 each time, but prepaying for an unlimited number of prescriptions is:

  • £29.10 for 3 months
  • £104 for 12 months (or 10 Direct Debit installments of £10.40)

You can find out how they work and buy one online.
